Detailed Description

This class is about reading the fits file, transforming the data array and displaying the result in various ways.

We can read a FITS or a DAT file, the result is then stored into a output file (binary or fits). The hypercube extracted is put into a larger hypercube of dimensions \( dim\_\nu \times 2^{n\_side} \times 2^{n\_side} \). We can then rebuild the hypercube using the results, print the mean values of the gaussian parameters or print the gaussian parameters map.

We read and write the FITS file using the library CCFits based on CFitsio.
